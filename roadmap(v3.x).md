# 🗺️ DotGhostBoard Roadmap — v3.x

> **Project:** DotGhostBoard  
> **Author:** FreeRave / DotSuite  
> **Created:** 2026-04-14

---

## 🧱 Principles for v3.x

- 🌍 **Boundless Reach** — Beyond local networks, embracing mesh architecture.
- 🔐 **E2EE by Default** — All shared data is encrypted end-to-end, no exceptions.
- 🤝 **Team-Ready** — Designed for multi-device and multi-user workflows.
- 🧩 **Extensible** — Community-driven plugins without bloating the core.

---

## 📅 v3.0.0 — Hydra `[Planned]`

> **Goal:** Mesh Syncing & Teams — Decentralized Sharing

### Features
- **Mesh Network Sync:** Sync across devices on the same network or via Tailscale/ZeroTier detection.
- **Shared Collections (E2EE):** Share specific encrypted folders with team members via E2EE group keys.
- **Conflict Resolution:** Smart handling of simultaneous edits or sync states across more than 2 devices.

---

## 📅 v3.1.0 — Kraken `[Planned]`

> **Goal:** Advanced P2P & Cross-Platform

### Features
- **WebRTC Transport:** P2P data transfer between devices — no relay server needed, NAT-piercing.
- **Cross-Platform Foundation:** Official Windows and macOS builds to support cross-OS sync.

---

## 📅 v3.2.0 — Phantom Plugins `[Planned]`

> **Goal:** Community & Extensions

### Features
- **Plugin System:** Let users write custom Python extensions to extend DotGhostBoard's behavior.
